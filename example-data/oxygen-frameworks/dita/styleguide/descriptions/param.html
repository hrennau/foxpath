<param>

The <param> (parameter) element specifies a set of values that may be required by an <object> at runtime. Any number of <param> elements may appear in the content of an <object> in any order, but must be placed at the start of the content of the enclosing object. This element is comparable to the XHMTL <param> element, and its attributes' semantics derive from their HTML definitions. For example, the @type attribute differs from the @type attribute on many other DITA elements.

Category: Body elements
